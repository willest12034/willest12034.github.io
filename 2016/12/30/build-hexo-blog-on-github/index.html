使用Hexo和Github建立Blog

Posted on 2016-12-30 |

什麼是Hexo

Hexo是一個基於NodeJS所建立的網誌框架。
它能快速幫你搭建好部落格，還有很多強大的外掛支援，並可用Markdown語法編寫文章。

開始使用Hexo

安裝

請先確定你已經安裝及設定好以下軟體：

  • Node.js
  • Git

若已經準備好以上軟體，現在就可以透過 npm 來安裝 Hexo

建立部落格應用

安裝完成後，就可以開始建立自己的部落格了

本地端啟動部落格

建立完成後，就可以開始測試我們的部落格

這時連結到localhost:4000，即可看到部落格囉

預設會在 localhost:4000

建立文章

確認部落格能正常運行後，我們來試著建立一篇文章吧

這時在我們新建的文章檔案會在如下位置中：

在 Github Page 上建立部落格

現在我們需要將部落格部署至一個網路空間，這樣才能讓別人看得到，
在考量多個方案後，我選擇免費的 Github Page。

但首先，我們需要一個 Github 帳號

建立Repository

在 Github 上建立一個新的 Repository

Repository Name 必須為 使用者名稱.github.io

接著將 Repository clone 到本機

把部落格丟上Github

現在我們準備把我們的部落格丟到Github上了

首先回到Hexo部落格資料夾底下

這時會在專案路徑下生成一個 public 資料夾

接著把 public 資料夾 內的所有東西丟到剛clone下來的 Repository
並切換到 Repository 資料夾 路徑下，把資料夾的內容 commit 後，push 到 Github

成功後等待數秒，就可以試著連到自己的網頁了

格式為 https:// 使用者名稱.github.io/
例如我的：https://willest12034.github.io/

若是有出現畫面，就代表成功囉！
